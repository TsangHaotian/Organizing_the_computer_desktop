# 桌面整理工具 - 新版本

## 🎉 新功能特性

### 📱 固定5列网格布局
- 所有格子固定为5个一排，整齐美观
- 支持垂直滚动，可以无限添加应用
- 默认创建4行5列共20个格子

### 🧭 现代化导航条
- **+ 添加应用**: 快速添加新的应用快捷方式
- **⚙️ 设置**: 打开设置界面
- **🔒 锁定/🔓 解锁**: 锁定窗口位置防止意外移动

### 🎯 智能拖拽功能
- **直接拖拽添加**: 将文件/应用拖到悬浮窗即可添加
- **优先空格子**: 自动添加到第一个空格子，不会覆盖已有应用
- **自动命名**: 根据文件名自动设置应用名称

### 🎨 图标布局优化
- **图标在上，文字在下**: 更符合桌面应用的视觉习惯
- **统一图标大小**: 所有图标保持一致的大小
- **可调节图标大小**: 在设置中可以调整图标大小（16px-128px）
- **自适应布局**: 根据窗口宽度自动调整格子大小
- **最小宽度保护**: 窗口最小宽度440px，确保5列布局正常显示
- **现代化样式**: 圆角边框，悬停效果，半透明背景

### 🎛️ 人性化设置界面
- **应用管理**: 可视化管理所有应用，支持添加、编辑、删除
- **外观设置**: 调整窗口透明度、大小、图标大小等
- **高级设置**: 锁定位置、重置设置等

## 🚀 使用方法

### 基本操作
1. **启动应用**: 运行 `python main.py`
2. **添加应用**: 
   - 方法1: 直接拖拽文件到悬浮窗
   - 方法2: 点击导航条的"+ 添加应用"按钮
3. **打开应用**: 点击格子中的图标即可启动
4. **管理应用**: 右键点击图标或使用设置界面

### 设置界面
- **应用管理**: 查看所有已添加的应用，支持编辑和删除
- **外观设置**: 
  - 调整透明度：让悬浮窗更好地融入桌面背景
  - 调整窗口大小：根据需要调整悬浮窗尺寸
  - **调整图标大小**：16px-128px范围内自由调节
- **高级设置**: 锁定窗口位置、重置所有设置

## 💡 使用技巧

1. **快速添加**: 直接从桌面拖拽应用到悬浮窗
2. **批量整理**: 将常用应用分类添加到不同分组
3. **自定义图标**: 在设置中为应用设置自定义图标
4. **调整透明度**: 根据桌面背景调整窗口透明度
5. **图标大小调节**: 根据个人喜好调整图标大小，小图标更紧凑，大图标更清晰

- **垂直布局**: 图标在上，文字在下，更符合用户习惯

### 配置管理
- **图标大小配置**: 支持16-128像素范围的图标大小调节
- **实时预览**: 设置变更立即生效
